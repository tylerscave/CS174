readMe.text contains information on setting up the 
Image Rating system on a new server.
Solves CS174 Hw3
@author Tyler Jones

1. Please put the hw3 folder and its subfolders into your servers document root
2. Create the imageRating database.
    a) Navigate to /hw3/src/configs/
    b) Open config.php and enter any specific database values for your system.
        1) Note these are set up to system defaults by default. edit as needed.
    b) Run the command "php CreateDB.php" from the terminal to create the database.
3. Create an account and start adding and rating images!!!
